# qr-telegram-bot
My first messenger bot. It turns any text into qr code.

## Installation

Install all requirements

## Usage

Clone the repo

```bash
git clone https://github.com/pischule/qr-telegram-bot.git
cd qr-telegram-bot
```

Put your token into '.env'

```bash
echo 'TOKEN=???????' > .env
```

Run the app!

```bash
docker-compose up -d
```
